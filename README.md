# NEWSAPP

Daily News Portal.

Vue3.js + Vuetify + Vuelidate + Vitest 


### .env
```
Please ensure to add valid NEWSAPI key to .env file
```

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run Vitest
```
npm run test 
```
